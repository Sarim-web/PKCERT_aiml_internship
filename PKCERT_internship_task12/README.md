# PKCERT Internship Task 12

This project contains a Jupyter notebook for clustering and dimensionality reduction using the IBM HR dataset.

## Structure

- notebooks/ - analysis notebooks
- data/ - dataset storage (optional, since the example can load from sklearn)
- src/ - reusable Python helpers if needed
- reports/ - exported figures and results

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook in the notebooks folder.
