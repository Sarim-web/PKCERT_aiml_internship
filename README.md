# PKCERT AI & Software Development Internship – Tasks 12–15

> **Advanced Machine Learning Projects**
>
> **Submitted by:** Sarim Ahmed

This repository contains the solutions for **Tasks 12–15** completed during the **PKCERT AI & Software Development Internship**. These tasks focus on advanced machine learning concepts, including model evaluation, handling imbalanced datasets, ensemble learning, model persistence, and developing complete end-to-end machine learning pipelines.

---

## 📌 Tasks Included

### 🔹 Task 12 – Model Evaluation & Performance Metrics
- Evaluated classification models using standard performance metrics.
- Generated Confusion Matrix and Classification Report.
- Calculated Precision, Recall, F1-Score, and Accuracy.
- Learned how different evaluation metrics are used for classification problems.

---

### 🔹 Task 13 – Advanced Model Evaluation & Handling Imbalanced Data
- Implemented ROC Curve and ROC-AUC analysis.
- Generated Learning Curves for model diagnostics.
- Addressed class imbalance using:
  - SMOTE (Synthetic Minority Over-sampling Technique)
  - Class Weighting
- Compared different imbalance handling techniques.

---

### 🔹 Task 14 – Ensemble Methods (Bagging & Boosting)
- Built ensemble learning models using:
  - Random Forest (Bagging)
  - XGBoost (Boosting)
- Compared model performance using classification metrics.
- Analyzed strengths and weaknesses of both ensemble techniques.
- Recommended the most suitable model based on experimental results.

---

### 🔹 Task 15 – Model Persistence & End-to-End ML Mini Project
- Developed a complete machine learning pipeline.
- Performed Exploratory Data Analysis (EDA).
- Applied preprocessing using **ColumnTransformer**.
- Trained a Logistic Regression classifier.
- Saved trained models using:
  - Pickle
  - Joblib
- Loaded and verified saved models.
- Built a reproducible workflow suitable for deployment.
